#  Starbucks India Sales & Revenue Dashboard

An interactive **Microsoft Excel dashboard** designed to analyze sales performance, customer behavior, and revenue trends across Starbucks stores in India using a **simulated dataset generated for data analytics and visualization practice**.

##  Key Features
- KPI Tracking (Revenue, Orders, Average Order Value, Ratings)
- Revenue by Store Type Analysis
- Top Cities by Revenue
- Monthly Revenue Trends
- Product Category Mix Analysis
- Interactive Slicers (State, City, Store Type, Month)

##  Key Insights
- Generated total revenue of **₹69.61 Cr** from **27.13 lakh orders**.
- Maintained an average order value of **₹276**.
- Achieved an average customer rating of **4.4/5**.
- **Airport stores** contributed the highest revenue among all store types.
- **Delhi and Chennai** emerged as top-performing cities by revenue.
- Revenue peaked during **April–June**, indicating stronger seasonal demand.
- **Coffee products** accounted for **44%** of total revenue.
- **Cold beverages** contributed **20%** of overall sales.
- Bakery and snack categories together contributed more than **25%** of category revenue.

##  Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Conditional Formatting
